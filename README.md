# ContextHub

A tool to generate AI context from project files, so you can quickly onboard any AI assistant with full project understanding.

## Features (Planned)
- Upload project files (ZIP)
- Automatic parsing of file structure and key files
- Generate a Markdown context summary
- Copy context to clipboard for use in AI chats
- Version history for projects
- GitHub integration (future)

## Tech Stack
- Backend: Node.js, Express, PostgreSQL
- Frontend: React, Tailwind CSS
- Deployment: Render (backend), Vercel (frontend), Supabase (database & storage)
